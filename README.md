# Chen Skills - Cypher 专属技能库

> AI Agent 能力沉淀与复用中心

## 目录结构

```
chen-skills/
├── ai-flavor-remover/     # AI 文章润色
├── financial-market/      # 金融市场分析
├── dev-tools/            # 开发工具集
├── writing-assist/       # 写作辅助
└── templates/            # 通用模板
```

## 使用方式

每个 Skill 包含：
- `SKILL.md` - 技能说明文档
- `*.py` 或脚本 - 可执行代码
- `README.md` - 使用指南

## 设计原则

1. **单一职责**：一个 Skill 只做一件事
2. **可复用性**：不依赖特定上下文
3. **渐进增强**：从简单开始，逐步完善
4. **文档优先**：先有文档，后有代码
