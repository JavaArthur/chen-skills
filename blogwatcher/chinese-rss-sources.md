## 使用命令

```bash
# 查看所有订阅
blogwatcher blogs

# 扫描更新
blogwatcher scan

# 查看文章列表
blogwatcher articles

# 标记已读
blogwatcher read 1

# 添加新源
blogwatcher add "名称" "RSS地址"

# 删除源
blogwatcher remove "名称"
```
