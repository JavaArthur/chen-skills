# Skill 模板

创建新 Skill 时，复制此模板并填写内容。

```markdown
# Skill: [技能名称]

> [一句话描述]

## 功能概述

[详细描述这个 Skill 能做什么]

## 使用场景

- [场景 1]
- [场景 2]
- [场景 3]

## 使用方法

### 基础用法

```bash
[命令/使用示例]
```

### 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| [参数名] | [类型] | [是/否] | [说明] |

## 示例

### 示例 1: [场景]

**输入:**
```
[输入示例]
```

**输出:**
```
[输出示例]
```

## 设计思路

[为什么会有这个 Skill？解决什么问题？]

## 依赖

- [依赖 1]
- [依赖 2]

## 版本

**Version:** [x.y.z]

**Changelog:**
- v[x.y.z]: [改动说明]
```

## 命名规范

1. **Skill 文件名**: 使用 kebab-case，如 `ai-flavor-remover`
2. **目录结构**:
   ```
   skill-name/
   ├── SKILL.md          # 技能说明（必须）
   ├── README.md         # 使用指南
   ├── main.py|sh        # 主程序
   ├── config/           # 配置文件
   └── tests/            # 测试用例
   ```

3. **版本号**: 遵循 SemVer (主版本.次版本.修订号)
